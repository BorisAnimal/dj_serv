по url [localhost]:[port]/users/check/ авторизация по логину, паролю через JSON (возвращает token)
по url /admin админка. Создайте supersuer командой в консоле. Показывает всю БД
по url /users/accounts/ пока возвращает dummy словарь (JSON). 


Как пользоваться: банальный django
импоритть env: myvenv\Script\activate 
pyhton manage.py runserver

чтобы создать ак админа
pyhton manage.py createsuperuser


(если какие команды не робят, ищите в инете опечатки)
